Асинхронное программирование в JavaScript

Описание проекта:
Этот проект — подборка задач, которые я выполнил для изучения и отработки навыков асинхронного программирования в JavaScript. 
Он охватывает использование callback-функций, Promises и async/await, а также обработку ошибок. 
Проект продемонстрирует мои знания асинхронных методов и подходов, необходимых для построения надёжного и производительного кода.

Список задач:
1. Обработка массива — Умножение каждого элемента массива на 2 с использованием callback-функции.
2. Фильтрация массива — Выбор только чётных чисел массива и передача результата через callback.
3. Последовательное выполнение задач — Использование цепочки .then() для последовательного вызова Promises.
4. Вероятностное выполнение задач — Реализация задачи с 50% шансом на успех, прекращение цепочки при ошибке.
5. Получение данных с использованием async/await — Имитация асинхронного запроса данных.
6. Обработка ошибок в async/await — Демонстрация обработки вероятностной ошибки.
7. Параллельное выполнение задач — Одновременное выполнение нескольких задач с использованием Promise.all.
8. Загрузка категорий и продуктов — Последовательная загрузка данных с задержкой и упорядоченный вывод.
