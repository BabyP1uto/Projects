Crypto Price Fetcher

Этот проект представляет собой приложение, которое получает текущие цены на криптовалюты с использованием API CoinGecko. Пользователь может ввести название криптовалюты и валюты, а приложение отобразит актуальную цену. В случае ошибок (например, криптовалюта не найдена или ошибка сервера) код автоматически попытается повторить запрос несколько раз.

Технологии

1. Node.js (используется для запуска кода на сервере)
2. Fetch API для отправки запросов к внешнему API
3. URLSearchParams для работы с параметрами запроса
4. Асинхронное программирование с использованием async/await
5. Обработка ошибок и повторные попытки при неудачных запросах

Особенности

-- Асинхронные запросы: проект использует асинхронные запросы через fetch, что позволяет эффективно работать с данными из внешнего API.
-- Обработка ошибок: предусмотрена обработка различных типов ошибок (например, не найденная криптовалюта или ошибка сервера).
-- Повторные попытки: в случае ошибки запрос повторяется несколько раз с случайной задержкой.
-- Множественные запросы: поддержка нескольких запросов для получения цен разных криптовалют в различных валютах.
-- Логирование: вывод информации о времени запроса и его результате.

Навыки, приобретенные в процессе разработки

1. Асинхронное программирование: опыт работы с Promise, async/await, обработка ошибок в асинхронных операциях.
2. Работа с внешними API: получение данных с API (в данном случае CoinGecko) с использованием параметров запроса.
3. Обработка ошибок: создание механизмов для повторных попыток запроса и обработки ошибок (например, 404 и 500 ошибки).
4. Тестирование отказоустойчивости: обеспечение корректности работы при сбоях с использованием ограничений на количество попыток.
5. Работа с параметрами URL: использование URLSearchParams для динамической генерации URL с параметрами.
6. Разработка с использованием Node.js: создание скриптов на серверной стороне для работы с API
